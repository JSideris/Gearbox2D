GearBox2D

High-Speed 2D Physics Engine powered by WebAssembly

Features

Blazing Fast Performance

Written in C++ and compiled to WebAssembly, GearBox2D delivers incredible speed with its data-oriented design and shared memory model.

Ideal for High-Frequency Updates

Perfect for simulations and applications that require frequent updates, like multiplayer synchronization with remote hosts.

Ease of Use

GearBox2D is easy to integrate and versatile, making it a great tool for developers of all skill levels.

Work in Progress

GearBox2D is currently in early alpha with limited functionality. Join the community and help shape its future!
